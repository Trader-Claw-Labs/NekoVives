# CLAUDE.md ‚Äî Trader Claw

## What is this project?
Fork of TraderClaw (traderclaw-labs/traderclaw). A Rust crypto trading agent
for EVM (Uniswap), Solana (Raydium/PumpFun), TON (STON.fi), and
Polymarket prediction markets.

Full specifications are in: trader_agent_research.docx

## Build & Test
```bash
# Full build (embeds web/dist into binary)
cargo build --release

# Rebuild web dashboard only
cd web && npm run build && cd ..

# Both steps required after any frontend change
cd web && npm run build && cd .. && cargo build --release

# Run
./target/release/trader-claw gateway

cargo test
cargo clippy -- -D warnings
docker compose up -d
```

## Binary
- Name: `trader-claw` (was `trader-agent` / `degen-agent`)
- Config dir: `~/.config/trader-claw/`
- Config file: `~/.traderclaw/config.toml`

## Architecture
Workspace crates:
- `.` (src/)                ‚Äî main binary, gateway, channels, tools, agent loop
- `crates/wallet-manager`  ‚Äî EVM BIP44, Solana ED25519, TON v4R2 ‚Äî AES-256-GCM + Argon2id keystore
- `crates/evm-trader`      ‚Äî Uniswap V2/V3/V4 via alloy + uniswap-sdk-core
- `crates/solana-trader`   ‚Äî PumpFun, Raydium via sol-trade-sdk
- `crates/ton-trader`      ‚Äî STON.fi via tonlib-rs
- `crates/polymarket-trader` ‚Äî Gamma + CLOB API, L1/L2 auth, WebSocket
- `crates/market-analyzer` ‚Äî TradingView Screener HTTP client (fetch_indicators, top_crypto_symbols)

## Web Dashboard (web/)
React + Vite + TanStack Query + Tailwind. Assets embedded into binary via rust-embed.
Rebuild: `cd web && npm run build` then `cargo build --release`.

Pages and routes:
- `/`                ‚Äî Dashboard (status cards, system health, market scanner widget)
- `/wallets`         ‚Äî Web3 Wallets (EVM ¬∑ Solana ¬∑ TON)
- `/polymarket`      ‚Äî Polymarket prediction market trading
- `/telegram`        ‚Äî Telegram bot config
- `/skills`          ‚Äî Cron strategies (scheduled jobs)
- `/chat`            ‚Äî Multi-chat parallel AI sessions
- `/tradingview`     ‚Äî TradingView Screener: live RSI, MACD, price table + active signals panel
- `/backtesting`     ‚Äî Strategy backtesting: .rhai script runner, metrics, worst trades, AI analysis
- `/settings/llm`    ‚Äî LLM provider/model config
- `/settings/config` ‚Äî Advanced config

## Gateway API Routes
All `/api/*` require Bearer token auth (pair via `POST /pair` with `X-Pairing-Code` header).
Public: `GET /health`, `GET /metrics`, `POST /pair`.

Key routes:
- `GET  /api/status`                   ‚Äî system overview
- `GET  /api/tradingview/scan`         ‚Äî TradingView Screener indicators (?symbols=BTCUSDT,ETHUSDT)
- `GET  /api/backtest/scripts`         ‚Äî list .rhai files from /scripts/
- `POST /api/backtest/run`             ‚Äî run backtest (Rhai engine, stub returns metrics)
- `GET  /api/wallets`                  ‚Äî list wallets
- `POST /api/wallets/create`           ‚Äî create wallet (EVM/Solana/TON)
- `GET  /api/polymarket/markets`       ‚Äî list markets
- `GET  /api/cron`                     ‚Äî list cron jobs
- `POST /api/cron`                     ‚Äî add cron job
- `GET  /api/memory`                   ‚Äî list memories

## Backtesting Engine (spec)
- `/scripts/`  ‚Äî .rhai strategy files generated by the agent
- `/data/`     ‚Äî Parquet historical data (crypto + Polymarket)
- `/results/`  ‚Äî JSON metric outputs for Claude to analyze
- Engine: Rust + Rhai (sandboxed, memory/time limits)
- Parallelism: Rayon for multi-param sweeps
- Metrics reported: PnL, Sharpe Ratio, Max Drawdown, Win Rate, 5 worst trades

## Pairing / Auth
Gateway requires pairing when `require_pairing = true` in config.
1. Start gateway ‚Äî one-time code printed to terminal
2. Open dashboard ‚Üí pairing modal appears automatically
3. Enter code ‚Üí `POST /pair` ‚Üí Bearer token saved to localStorage
4. Token persisted to `config.toml` for future restarts

## Security ‚Äî NEVER violate these
- NEVER log private keys, mnemonics, or Polymarket L2 credentials
- ALWAYS encrypt secrets at rest (AES-256-GCM + Argon2id)
- ALWAYS validate amounts before signing any tx or order
- Polymarket wallet must be a dedicated Polygon wallet
- Rhai scripts run sandboxed ‚Äî enforce memory + execution time limits
